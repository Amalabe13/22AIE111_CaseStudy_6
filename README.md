## Problem Description
* The Bank Management System is designed to manage basic banking operations in a simple and organized way.
* It helps in handling customer details, account management, and transaction processing such as deposit, withdrawal, and balance checking.
* The system ensures proper record-keeping of all transactions and provides a structured way to manage banking activities efficiently.
## How to Run the Code


### Step 1: Clone the Repository

```bash
git clone https://github.com/Amalabe13/22AIE111_CaseStudy_6.git
```

### Step 2: Open the Project Folder

```bash
cd 22AIE111_CaseStudy_6
```

### Step 3: Open in VS Code

```bash
code .
```

### Step 4: Compile the Java Files

Open terminal in VS Code and run:

```bash
javac src/*.java
```

### Step 5: Run the Project

```bash
java -cp src B
```
